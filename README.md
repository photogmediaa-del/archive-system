# نظام الأرشفة الإلكترونية - مجلس النواب العراقي

نظام أرشفة إلكترونية احترافي مبني بـ HTML/CSS/JavaScript، يعمل بالكامل في المتصفح بدون خادم.

## المزايا

- 🔐 **نظام مستخدمين** بأكواد وكلمات مرور تلقائية
- 📁 **أرشفة الوثائق** مع رفع ملفات أو من السكنر/الكاميرا
- 🔢 **أكواد متسلسلة** لكل وثيقة (مرتبطة بالمستخدم)
- 📋 **حقول الصادر/الوارد/الكتاب** للوثائق الرسمية
- 🗂️ **الأرشيف بالتاريخ** (سنة → شهر → يوم)
- 🔍 **البحث المتقدم** في كل الحقول
- 🖨️ **طباعة الوثائق** بقالب رسمي
- 💾 **النسخ الاحتياطي** والاستيراد
- 📋 **سجل النشاط** الكامل
- 🎨 **شعار قابل للتخصيص** يُحفظ في النظام

## التشغيل

افتح `index.html` في المتصفح مباشرة.

**الحساب الافتراضي**: `admin` / `admin123`

## التقنيات

- HTML5 / CSS3 / Vanilla JavaScript
- SQLite (sql.js) للقاعدة
- LocalStorage للحفظ المحلي
- IBM Plex Sans Arabic للخط
- SVG icons inline

## النسخة القادمة

نسخة Electron (.exe) مع:
- قاعدة بيانات مركزية على الشبكة المحلية
- دعم سكنر TWAIN مباشر
- مزامنة بين المستخدمين
